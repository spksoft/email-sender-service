# email-sender-service

this service was generate from @spksoft/nodejs-starter-generator

### Install dependency
```
yarn
```

### .env
```
APP_NAME=email-sender-service
PORT=8080
SENDGRID_API_KEY=
SPARKPOST_API_KEY=
MONGO_URL=
```

### Run a test
```
yarn test-unit
```

### Run a development server
```
yarn start-dev
```

### Compile & Run a production
```
yarn build
node build/index.js
```
